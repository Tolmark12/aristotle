#Movie


#### layers
An array of layers
```yaml
movie:
  layers:
    - {depth: 0, content: first-layer.svg }
    - {depth: 1, content: second-layer.jpg }
    - {depth: 2, content: third-layer.json }
```
#### zoom
The zoom amount and point to zoom into. (ammount, xorigin, yorigin)
```yaml
movie:
  zoom: [2, 300, 400]
```
